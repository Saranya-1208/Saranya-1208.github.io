# Digital India: Measuring the Digital Transformation

Data visualization project examining India's digital infrastructure transformation through internet connectivity, Aadhaar digital identity, and UPI payments.

**Live Site:** https://saranya-1208.github.io/project/

## Key Features
- 9 interactive visualizations
- API integration (5G government data)
- Web scraping (McKinsey PDF extraction)
- Machine learning forecasting
- 250K transaction behavioral analysis

## Data Sources
- World Bank World Development Indicators (internet penetration)
- National Payments Corporation of India (UPI transactions)
- India Open Data Portal (5G infrastructure)
- Ministry of Electronics (Digital Villages pilot program)
- Aadhaar enrollment records (236K enrollments across 5,000 pincodes)

## Technical Stack
- **Data Processing:** Python (pandas, scikit-learn)
- **Visualization:** Vega-Lite
- **Analysis:** Google Colab notebooks

## Visualizations
1. Global internet comparison (2010-2024)
2. McKinsey digital adoption ranking
3. UPI explosion timeline (116 months)
4. Transaction timing heatmap (250K transactions)
5. UPI per capita state map (interactive dropdown)
6. Digital village services analysis
7. Aadhaar enrollment diverging bar chart
8. 5G infrastructure coverage map
9. Machine learning UPI forecast (2026)
